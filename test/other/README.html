  • c : 元数据list
  • tx: 文字段
  • li: 艺术家、歌手头像图url

  2. 歌词行显示总时长(毫秒)
  3. 逐字显示开始时间戳 (毫秒)
  4. 逐字显示时长 (厘秒/0.01s)
  5. 未知
  6. 文字
